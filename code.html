Code

Here are some open source projects I have played some role in.

  • Sync Kit

    Sync Kit optimizes web application hosting by persistently caching both relational data and static resources in the client browser, allowing the client to take over much of the processing load associated with web hosting.

  • Datapress

    Datapress is a tool for incorporating structured data into the blogging workflow. It helps bloggers link to, visualize, and share their data sets with the world.

  • Wibit

    Wibit enables Semantic MediaWiki users to easily visualize and browse their knowledge repository with Exhibit.

  • Embedded JavaScript

    Embedded JS (now maintained by Jupiter IT) is an open source JavaScript template library I wrote. It is currently in use in the JavaScript MVC library and the Aptana Cloud service.

  • Snoggle

    Snoggle is a graphical ontology mapper that exports mappings as Semantic Web Rule Language (SWRL) rules.
