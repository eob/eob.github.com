Code

Here are some open source projects I have played some role in.

  • Bookmarklet Maker

    Bookmarklet Maker is a simple tool to help you make Bookmarklets that bootstrap themselves via external Javascript and CSS, enabling you to provide users an easy way to attach externally hosted (and updated) capability to any web site.

  • Researchy

    Researchy is a Batman utility belt for your Scala research code. It includes a "lab notebook" to save files and data to, timers, and git integration.
