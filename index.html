Ted Benson

Hello.

I am a Ph.D. student at MIT CSAIL advised by David Karger. My research is driven by a desire to enable us to create rich, data-driven web content with the same ease that we consume it.

My thesis work is on Cascading Tree Sheets, which does for web structure what CSS did for style. I've also done a bit of natural language processing work at MIT and Google, which you're using whenever you talk to your Android phone. Prior to coming to MIT, I did R&D at BBN Technologies.
