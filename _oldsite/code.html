Here are some links to various open-source projects I've been a part of.

Web Toolkits

  • Cascading Treesheets (CTS)

    Cascading Treesheets does for web structure what CSS does for web style. It is a lightweight declarative annotation layer which helps you merge, split, and remap web content.

  • Embedded Javascript (EJS)

    Embedded Javascript is an open source, client-side JavaScript template library I wrote in 2007. Looking back, I believe it was one of the first in-browser template libraries created.

    EJS was used widely and embedded in other frameworks (such as JavaScript MVC and Aptana's offerings). The official site for EJS is maintained by Jupiter IT.

  • Sync Kit

    Sync Kit (2009) caches the uncacheable -- dynamic, user-sensitive data -- in the client browser. It optimizes web application hosting by persistently caching both relational data and static resources in the client browser, allowing the client to take over much of the processing load associated with web hosting. Here's a sketch of how it works: the nth time you visit a blog, your browser only needs to download a "data diff" against the n-1th time you visited the blog.

    Services such as Cloudflare Railgun (2012) and Meteor are beginning to implement this style of optimization.

  • Datapress

    Datapress is a tool for incorporating structured data into the blogging workflow. It helps bloggers link to, visualize, and share their data sets with the world.

Odds and Ends

  • English to Japanese Transliteration Model

    A fun class project building a Hidden Markov Model to transliterate English phrases into Katakana.

  • Researchy

    Researchy is a Batman utility belt for your Scala research code. It includes a "lab notebook" to save files and data to, timers, and git integration.

  • Snoggle

    Snoggle is a graphical ontology mapper that exports mappings as Semantic Web Rule Language (SWRL) rules. Note: from a long time ago & no longer maintained.
